Lista

Este projeto consiste em uma aplicação web que permite aos usuários gerenciar suas atividades diárias de forma simples e eficiente.
Com a capacidade de adicionar novas tarefas, marcar aquelas que foram concluídas e remover itens desnecessários, a lista ajuda a manter a organização


Clima

Este projeto é uma aplicação web que fornece informações atualizadas sobre o clima em diferentes locais ao redor do mundo. Os usuários podem buscar
dados meteorológicos para qualquer cidade, incluindo a temperatura atual, condições climáticas (como se está chovendo ou ensolarado)


Cartão Direct Prime

Este projeto consiste na criação de um site dedicado ao nosso grupo do projeto final do Senai. O objetivo é apresentar cada integrante da equipe, destacando suas contribuições e competências.
No site, você encontrará cartões de apresentação de cada membro, que incluem uma breve biografia, as responsabilidades específicas que desempenharam no projeto e meios de contato.
Assim, é possível conhecer como cada um contribuiu para o sucesso do trabalho, desde a pesquisa inicial até a implementação final.
